NodeRCは、RealityCapture / RealityScanのCLIコマンド用の非公式なノードベースのビジュアルインターフェイスおよびエディタです。PythonとPyQt5を使用して開発されており、インタラクティブなキャンバス上でコマンドノードを視覚的に接続・管理し、ワークフロー自動化のための使いやすいインターフェイスを提供します。

## 特徴

- **インタラクティブキャンバス:** パンとズームをサポートする無限のワークスペース。
- **ノードアーキテクチャ:** 着信および発信接続（ソケット）をサポートするさまざまなタイプのノード。
- **動的接続:** 実行ポートとデータポートの視覚的なリンク。
- **構成システム:** 集中構成ファイルを介した色、サイズ、スタイルのカスタマイズ。
- **検索メニュー:** キャンバスに新しいノードをすばやく追加するための便利なメニュー。

## 要件

- Python 3.7+
- PyQt5

## インストール

1. リポジトリのクローン:

   ```bash
   git clone <リポジトリのURL>
   cd nodeRC
   ```

2. 依存関係のインストール:
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

エディタを起動するには、以下を実行します:

```bash
python nodeRC.py
```

### 操作方法とショートカット

- **キャンバスのパン（移動）:** **マウスの中ボタン（ホイールボタン）**を長押ししながらドラッグします。
- **ズーム（拡大・縮小）:** **マウスホイール**を前後にスクロールします。
- **ノードの追加:** **スペースキー**を押すか、キャンバスの何もない部分を**右クリック**して検索メニューを開きます。コマンドまたはパラメータノードを選択して作成します。
- **ソケットの接続:** 出力ソケットから互換性のある入力ソケットまでドラッグして接続線を引きます。
  - ソケットからドラッグしてキャンバスの何もない場所で離すと、検索メニューが開き、選択した新規ノードが自動接続されます。
- **ノード/接続の削除:** 対象を選択して **Delete** キーを押します。
- **ノードのグループ化:** 複数ノードを選択して **Ctrl + G** を押すと、論理グループフレームにまとめられます。
- **ノードの複製:** 対象を選択して **Ctrl + D** を押します。
- **コピー＆ペースト:** 対象を選択して **Ctrl + C** でコピーし、カーソル位置で **Ctrl + V** を押して貼り付けます。
- **元に戻す / やり直し:** **Ctrl + Z** で元に戻し、**Ctrl + Y**（または **Ctrl + Shift + Z**）でやり直します。
- **すべて選択:** **Ctrl + A** を押します。
- **グリッドの表示切替:** **G** キーを押してグリッドの表示/非表示を切り替えます。
- **表示のフィット:** **F** キーを押すと、選択したノード（未選択の場合は全ノード）に画面をフィットさせます。
- **ノード名の変更:** パラメータノードを選択し、**F2** キーを押します。
- **全画面表示:** **F11** キーを押して切り替えます。

### 実行フロー

1. **チェーンの開始点:** キャンバス上には常に `> START` ノードが存在します。
2. **コマンドノードの追加:** **スペースキー**で `-addFolder`、`-align` などのコマンドノードを追加します。
3. **実行経路の接続:** `> START` ノードの出力ソケットから順に、コマンドノードの矢印形状の実行ソケットを接続します。
4. **パラメータの設定:** 検索メニューからパラメータノード（String、Integer、Float、File/Dir Path など）を追加し、その出力ソケットをコマンドノードの入力ソケットへ接続します。
5. **実行:** `> START` ノード内の **> Launch** ボタンをクリックすると、接続されたコマンドチェーンが RealityCapture で実行されます。


## プロジェクト構造

- `nodeRC.py` - メインエントリポイント。
- `canvas.py` - メインエディタウィンドウのロジック。
- `scene.py` - キャンバスシーンおよびビジュアルアイテムのイベント処理。
- `view.py` - グラフィックビューのロジック、パン、ズーム。
- `nodes_base.py` - ノード、ソケット、接続のベースクラス。
- `nodes_concrete.py` - 特化ノード（Start、Command、Parameterノード）の具体的な実装。
- `configuration.py` - スタイル、UI設定、ショートカットキーの一元化された設定ファイル。
- `search_menu.py` - ノードを生成するためのオートコンプリート検索ダイアログ。
- `diagnostics.py` - 例外処理およびエラーログ出力。
- `rc_documentation_extractor.py` - ローカルのRealityCaptureドキュメントからコマンドデータベースを構築するユーティリティ。
