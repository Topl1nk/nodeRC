مشروع NodeRC هو محرر مرئي يعتمد على العقد (Nodes) تمت برمجته بلغة Python باستخدام PyQt5. يتيح لك المشروع إنشاء وتوصيل وإدارة العقد على لوحة تفاعلية، مما يوفر واجهة سهلة الاستخدام للبرمجة المرئية أو بناء المخططات المنطقية.

## الميزات

- **لوحة تفاعلية (Canvas):** مساحة عمل غير محدودة تدعم التحريك والتكبير.
- **هيكلية العقد:** أنواع مختلفة من العقد تدعم الاتصالات الواردة والصادرة (sockets).
- **اتصالات ديناميكية:** ربط مرئي لمنافذ التنفيذ ومنافذ البيانات.
- **نظام التكوين:** ألوان وأحجام وأنماط قابلة للتخصيص عبر ملف تكوين مركزي.
- **قائمة البحث:** قائمة مريحة لإضافة العقد الجديدة بسرعة إلى اللوحة.

## المتطلبات

- Python 3.7+
- PyQt5

## التثبيت

1. استنساخ المستودع:

   ```bash
   git clone <رابط_المستودع>
   cd nodeRC
   ```

2. تثبيت التبعيات:
   ```bash
   pip install -r requirements.txt
   ```

## الاستخدام

لبدء المحرر، قم بتشغيل:

```bash
python nodeRC.py
```

## هيكل المشروع

- `nodeRC.py` - نقطة الدخول الرئيسية.
- `canvas.py` - منطق اللوحة التفاعلية وإدارة الرسم البياني.
- `nodes.py` - الفئات الأساسية والمتخصصة للعقد والاتصالات.
- `configuration.py` - ملف التكوين (الألوان والأنماط ومعلمات واجهة المستخدم).
- `search_menu.py` - مربع حوار للبحث وإضافة العقد.
- `diagnostics.py` - التسجيل ومعالجة الاستثناءات.
- `rc_documentation_extractor.py` - أداة لاستخراج وثائق الأوامر.
