مشروع NodeRC هو واجهة مرئية ومحرر غير رسمي يعتمد على العقد (Nodes) لأوامر واجهة سطر الأوامر (CLI) لبرنامجي RealityCapture / RealityScan. تمت برمجة المشروع بلغة Python باستخدام PyQt5، ويتيح لك توصيل وإدارة عقد الأوامر بشكل مرئي على لوحة تفاعلية، مما يوفر واجهة سهلة الاستخدام لأتمتة سير العمل.

## الميزات

- **لوحة تفاعلية (Canvas):** مساحة عمل غير محدودة تدعم التحريك والتكبير.
- **هيكلية العقد:** أنواع مختلفة من العقد تدعم الاتصالات الواردة والصادرة (sockets).
- **اتصالات ديناميكية:** ربط مرئي لمنافذ التنفيذ ومنافذ البيانات.
- **نظام التكوين:** ألوان وأحجام وأنماط قابلة للتخصيص عبر ملف تكوين مركزي.
- **قائمة البحث:** قائمة مريحة لإضافة العقد الجديدة بسرعة إلى اللوحة.

## المتطلبات

- Python 3.7+
- PyQt5

## التثبيت

1. استنساخ المستودع:

   ```bash
   git clone <رابط_المستودع>
   cd nodeRC
   ```

2. تثبيت التبعيات:
   ```bash
   pip install -r requirements.txt
   ```

## الاستخدام

لبدء المحرر، قم بتشغيل:

```bash
python nodeRC.py
```

### عناصر التحكم والاختصارات

- **تحريك اللوحة:** اضغط مع الاستمرار على **زر الفأرة الأوسط (MMB)** واسحب.
- **التكبير والتصغير:** قم بتمرير **عجلة الفأرة**.
- **إضافة عقدة:** اضغط على مفتاح **المسافة (Space)** أو **انقر بزر الفأرة الأيمن** على مساحة فارغة في اللوحة لفتح قائمة البحث، ثم اختر العقدة المطلوبة.
- **توصيل المقابس (Sockets):** اسحب خطاً من مقبس الإخراج إلى مقبس إدخال متوافق.
  - يؤدي سحب خط من المقبس وإفلاته في مساحة فارغة إلى فتح قائمة البحث تلقائياً لإضافة عقدة جديدة وتوصيلها.
- **حذف عقدة/اتصال:** حدد العنصر واضغط على مفتاح **Delete**.
- **تجميع العقد:** حدد العقد المراد تجميعها واضغط على **Ctrl + G** لوضعها داخل إطار منطقي.
- **تكرار العقد:** حدد العقد واضغط على **Ctrl + D**.
- **النسخ واللصق:** حدد العقد واضغط على **Ctrl + C** للنسخ، و **Ctrl + V** للصق عند موقع المؤشر.
- **التراجع / إعادة التراجع:** اضغط على **Ctrl + Z** للتراجع و **Ctrl + Y** (أو **Ctrl + Shift + Z**) لإعادة التراجع.
- **تحديد الكل:** اضغط على **Ctrl + A**.
- **تبديل الشبكة:** اضغط على المفتاح **G** لإظهار أو إخفاء الشبكة.
- **ملائمة المنظور:** اضغط على المفتاح **F** لتركيز الرؤية على العقد المحددة (أو كل العقد إذا لم يكن هناك تحديد).
- **إعادة تسمية العقدة:** حدد عقدة معلمات واضغط على **F2**.
- **ملء الشاشة:** اضغط على **F11** لتبديل وضع ملء الشاشة.

### سير عمل التنفيذ

1. **بدء السلسلة:** عقدة البدء `> START` متواجدة دائماً على اللوحة.
2. **إضافة عقد الأوامر:** اضغط على **المسافة** وأضف العقد المطلوبة (مثل `-addFolder` و `-align` وغيرها).
3. **ربط مسار التنفيذ:** قم بتوصيل مقابس التنفيذ (على شكل سهم) بالتتابع بدءاً من مخرج عقدة `> START`.
4. **تكوين المعلمات:** أضف عقد المعلمات (String, Integer, Float, File/Dir Path) من قائمة البحث وقم بتوصيل مخرجاتها بمقابس إدخال عقد الأوامر.
5. **التشغيل:** انقر فوق زر **> Launch** في عقدة `> START` لتنفيذ سلسلة الأوامر في برنامج RealityCapture.


## هيكل المشروع

- `nodeRC.py` - نقطة الدخول الرئيسية.
- `canvas.py` - منطق نافذة المحرر الرئيسية.
- `scene.py` - مشهد اللوحة ومعالجة أحداث العناصر المرئية.
- `view.py` - منطق عرض الرسوم والتحريك والتكبير.
- `nodes_base.py` - الفئات الأساسية للعقد والمقابس والاتصالات.
- `nodes_concrete.py` - التنفيذ الفعلي للعقد المتخصصة (عقد البدء والأوامر والمعلمات).
- `configuration.py` - المصدر الموحد للأنماط وإعدادات واجهة المستخدم واختصارات لوحة المفاتيح.
- `search_menu.py` - مربع حوار البحث مع الإكمال التلقائي لإضافة العقد.
- `diagnostics.py` - معالج الاستثناءات وتسجيل الأخطاء.
- `rc_documentation_extractor.py` - أداة لبناء قاعدة بيانات الأوامر من وثائق RealityCapture المحلية.
