NodeRC, RealityCapture / RealityScan CLI कमांड्स के लिए एक अनौपचारिक नोड-आधारित विज़ुअल इंटरफ़ेस और संपादक है। पायथन और PyQt5 का उपयोग करके विकसित किया गया यह प्रोजेक्ट, आपको एक इंटरैक्टिव कैनवास पर कमांड नोड्स को विज़ुअली जोड़ने और प्रबंधित करने की अनुमति देता है, जिससे वर्कफ़्लो ऑटोमेशन के लिए उपयोगकर्ता के अनुकूल इंटरफ़ेस मिलता है।

## विशेषताएँ

- **इंटरएक्टिव कैनवास:** पैनिंग और ज़ूमिंग समर्थन के साथ एक अनंत कार्यक्षेत्र।
- **नोड आर्किटेक्चर:** इनकमिंग और आउटगोइंग कनेक्शन (सॉकेट्स) का समर्थन करने वाले विभिन्न प्रकार के नोड्स।
- **डायनेमिक कनेक्शन्स:** निष्पादन पोर्ट्स और डेटा पोर्ट्स का विज़ुअल लिंकिंग।
- **कॉन्फ़िगरेशन सिस्टम:** एक केंद्रीकृत कॉन्फ़िगरेशन फ़ाइल के माध्यम से अनुकूलन योग्य रंग, आकार और शैलियाँ।
- **खोज मेनू:** कैनवास में नए नोड्स को जल्दी से जोड़ने के लिए एक सुविधाजनक मेनू।

## आवश्यकताएं

- Python 3.7+
- PyQt5

## इंस्टालेशन

1. रिपॉजिटरी क्लोन करें:

   ```bash
   git clone <रिपॉजिटरी_का_URL>
   cd nodeRC
   ```

2. निर्भरताएं स्थापित करें:
   ```bash
   pip install -r requirements.txt
   ```

## उपयोग

संपादक शुरू करने के लिए, चलाएं:

```bash
python nodeRC.py
```

### नियंत्रण और शॉर्टकट

- **कैनवास घुमाना (पैन):** **माउस के मध्य बटन (MMB)** को दबाए रखें और ड्रैग करें।
- **ज़ूम:** **माउस व्हील** को स्क्रॉल करें।
- **नोड जोड़ें:** खोज मेनू खोलने के लिए **स्पेस** दबाएं या खाली स्थान पर **दायां क्लिक** करें। कर्सर की स्थिति पर कमांड या पैरामीटर नोड बनाने के लिए उसे चुनें।
- **सॉकेट कनेक्ट करें:** किसी आउटपुट सॉकेट से संगत इनपुट सॉकेट तक लाइन ड्रैग करें।
  - सॉकेट से लाइन ड्रैग करके खाली कैनवास स्थान पर छोड़ने पर खोज मेनू खुल जाता है, जिससे नए नोड को चुनकर ऑटो-कनेक्ट किया जा सकता है।
- **नोड/कनेक्शन हटाएं:** आइटम का चयन करें और **Delete** कुंजी दबाएं।
- **नोड्स को समूहीकृत (ग्रुप) करना:** नोड्स का चयन करें और उन्हें एक फ्रेम में रखने के लिए **Ctrl + G** दबाएं।
- **नोड्स की नकल (डुप्लिकेट) बनाना:** नोड्स का चयन करें और **Ctrl + D** दबाएं।
- **कॉपी / पेस्ट:** नोड्स का चयन करें और कॉपी करने के लिए **Ctrl + C** दबाएं, और कर्सर की स्थिति पर पेस्ट करने के लिए **Ctrl + V** दबाएं।
- **पूर्ववत (अनडू) / रीडू:** अनडू करने के लिए **Ctrl + Z** और रीडू करने के लिए **Ctrl + Y** (या **Ctrl + Shift + Z**) दबाएं।
- **सभी का चयन करें:** **Ctrl + A** दबाएं।
- **ग्रिड टॉगल करें:** ग्रिड दृश्यता को चालू/बंद करने के लिए **G** कुंजी दबाएं।
- **व्यू फिट करें:** चयनित नोड्स पर ध्यान केंद्रित करने के लिए **F** कुंजी दबाएं (यदि कोई चयनित नहीं है तो सभी नोड्स पर)।
- **नोड का नाम बदलें:** एक पैरामीटर नोड का चयन करें और **F2** दबाएं।
- **फुलस्क्रीन:** फुलस्क्रीन मोड चालू/बंद करने के लिए **F11** दबाएं।

### निष्पादन कार्यप्रवाह (Execution Workflow)

1. **श्रृंखला आरंभ करें:** `> START` नोड हमेशा कैनवास पर मौजूद रहता है।
2. **कमांड नोड्स जोड़ें:** **स्पेस** दबाएं और कमांड्स जोड़ें (जैसे, `-addFolder`, `-align`, आदि)।
3. **निष्पादन पथ लिंक करें:** `> START` नोड के आउटपुट से शुरू करते हुए कमांड नोड्स के तीर के आकार के निष्पादन सॉकेट्स को क्रम से जोड़ें।
4. **पैरामीटर कॉन्फ़िगर करें:** खोज मेनू के माध्यम से पैरामीटर नोड्स (String, Integer, Float, File/Dir Path) जोड़ें और उनके आउटपुट को कमांड नोड्स के इनपुट सॉकेट्स से जोड़ें।
5. **चलाएं (Run):** RealityCapture में कमांड श्रृंखला को निष्पादित करने के लिए `> START` नोड पर मौजूद **> Launch** बटन पर क्लिक करें।


## प्रोजेक्ट संरचना

- `nodeRC.py` - मुख्य प्रविष्टि बिंदु।
- `canvas.py` - मुख्य संपादक विंडो लॉजिक।
- `scene.py` - कैनवास दृश्य (सीन) और दृश्य तत्वों के ईवेंट हैंडलिंग।
- `view.py` - ग्राफिक व्यू लॉजिक, पैनिंग और ज़ूमिंग।
- `nodes_base.py` - नोड्स, सॉकेट्स और कनेक्शन के लिए बेस कक्षाएं।
- `nodes_concrete.py` - विशेष नोड्स (Start, Command, Parameter नोड्स) के ठोस कार्यान्वयन।
- `configuration.py` - शैलियों, UI सेटिंग्स और शॉर्टकट कुंजियों के लिए एकल स्रोत (Single Source of Truth)।
- `search_menu.py` - नोड्स बनाने के लिए स्वतः पूर्ण (autocomplete) खोज संवाद।
- `diagnostics.py` - अपवाद हैंडलर और त्रुटि लॉगिंग।
- `rc_documentation_extractor.py` - स्थानीय RealityCapture दस्तावेज़ीकरण से कमांड डेटाबेस बनाने की उपयोगिता।
